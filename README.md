### Instructions

1. To run the chatbot, you need an [OutSytems account](https://www.outsystems.com/Platform/Signup).
2. To get your **OutSystems Personal environment URL**.
3. To download the [Service Studio](https://www.outsystems.com/Portal/ServiceStudioTutorial?IsFreeEdition=False&Uuid=4136fd4f-bb21-4778-9b0c-9bffdbddf13b&DevHostname=Tu4gYoDjVr%2fnVf78LyifGX%2f%2feRIC%2fmy5E%2fCz95N%2bRmsqLtYr0eHJ2TvqGYajIYTlldzsVWplWMKH7Tua4%2ftcqrxxtPDvRoPSpNgWv6eApR4dre07RC5LXlkM%2fo4HcIbf&OpenSS=True).
4. To get an [OpenAI Key](https://platform.openai.com/api-keys).
5. To publish and run the chatbot.
